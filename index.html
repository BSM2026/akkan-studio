ABOUT THE MAKER

Handmade by Øzkan Akkan

Akkan Studio is built around unique handcrafted objects in wood. Each work is made by hand, with patience, detail and an intention to create something that cannot be found everywhere.

How it works

Every piece is made once

01 · Created by hand

No mass production. Every object is shaped, assembled and finished individually.

02 · Presented as available

Each object is published as part of the current collection.

03 · Sold once

When sold, the object is removed or marked as sold. The same piece will not be reproduced.
